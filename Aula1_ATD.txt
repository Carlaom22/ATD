1. Pretende-se analisar o sinal de tempo contínuo  x1(t)  = A1 sen(Wat)cos(Wbt) + A2 cos(Wct)^2

A1 = 2(mod(4,2) + 1),  A2 = 6(mod(4,2) + 1)
Wa = mod(4,5)+2, Wb = mod(4,7)+7, Wx = mod(4,9)+1

1.1 Obter a expressão do sinal de tempo discreto x1[n] que resulta de x1(t) usando t=nTa , em que Ts representa o período de amostragem com que o sinal de tempo contínuo x1(t) é amostrado.

Script 

% Dados fornecidos
A1 = 2*(mod(4,2) + 1);
A2 = 6*(mod(4,2) + 1);
Wa = mod(4,5) + 2;
Wb = mod(4,7) + 7;
Wc = mod(4,9) + 1;

% Frequência máxima presente no sinal contínuo
W_max = max([Wa, Wb, Wc]);

% Frequência de amostragem (utilizando o critério de Nyquist)
f_s = 2 * W_max;

% Período de amostragem
T_s = 1 / f_s;

% Definindo o intervalo de tempo discreto
n = 0:1:10; % ajuste o intervalo conforme necessário

% Amostragem do sinal contínuo
t = n * T_s;
x1_t = A1*sin(Wa*t).*cos(Wb*t) + A2*cos(Wc*t).^2;

% Sinal de tempo discreto resultante
x1_n = x1_t;

% Plotando o sinal discreto
stem(n, x1_n);
xlabel('n');
ylabel('x1[n]');
title('Sinal Discreto x1[n]');
grid on;
